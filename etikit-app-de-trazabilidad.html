Tutorial de la Aplicación Etik-it

Paso 1: Descarga de la Aplicación

Descarga la aplicación Etik-it desde Google Play o Apple Store. Acepta los permisos necesarios, incluyendo la ubicación geográfica.

Paso 2: Iniciación de la Aplicación

Al abrir la aplicación, aparecerá un mensaje indicando que realices la lectura.

Paso 3: Captura de Código QR

La cámara se abrirá automáticamente para capturar el código QR.

Paso 4: Ejemplo de Código

En la plataforma se incluirá la información relevante del producto o servicio y se generará un código encriptado que solo se puede leer con la aplicación Etik-it.

Paso 5: Lectura del Código

Utiliza la cámara para leer el código QR. Si el código está caducado, la aplicación lo indicará.

Paso 6: Producto Comprado

La plataforma muestra ejemplos de productos comprados, con un dashboard para gestionar categorías, productos, lotes, y más.

Paso 7: Generación de Códigos Únicos

Se pueden generar colas para impresión y crear archivos con códigos únicos encriptados.

Paso 8: Ejemplos de Uso

Los códigos pueden ser utilizados para permisos de trabajo, medicamentos, documentos notariales, diplomas, baterías de vehículos, etc.

Escríbenos y pide una prueba de Etikit

La aplicación Etik-it es una herramienta potente para la gestión y verificación de productos y servicios a través de códigos QR encriptados.
